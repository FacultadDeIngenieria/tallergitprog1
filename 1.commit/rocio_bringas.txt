1.1.1 Para iniciar un repositorio local en git se debe ejecutar el comando git init 
1.1.2 Para que un directorio deje de ser ejecutado por git se debe ejecutar el comando git rm 
1.1.3 Si, al agregar un archivo a un directorio que ya está siendo controlado por git, automaticamente esta siendo controlado por git 
1.1.4 Para agregar un archivo al repositorio local se debe ejectutar el comando git add, y para agregar un archivo especifico git add "nombre_archivo"
1.1.5 Para determinar que archivos fueron modificados se puede utilizar el comando git status y mostrara todos los archivos que se modificaron
1.1.6 Para hacer un commit se utiliza el comando; git commit -m "mensaje descriptivo"
1.1.7 Un commit es similar a un registro que muestra todas las modificaciones de un archivo desde su creacion, ayuda a entender que que cambios se hicieron y el porque de los mismos para que sea mas facil leerlo despues
