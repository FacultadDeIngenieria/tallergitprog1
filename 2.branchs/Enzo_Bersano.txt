1. un branch es una rama o division del codigo en git
2. pueden ser utiles los branches porque son de gran utilidad para que varios programadores puedan trabajar en el codigo de un proyecto al mismo tiempo
3.para crear un branch tenemos que utilizar el comando: git branch nombre de la branch
4. para cambiar de una branch a otra, debemos haer uso del comando: git branch -a y luego git checkout (nombre de la otra branch). 
5. para eliminar una branch, usamos el comando: git branch -d (nombre de branch a eliminar)
6. para crear una nueva branch y cambiar a esa: git switch -c 
7. un merge es la combinacion de branches
8. para realizar un merge, se hace uso del comando: git merge
